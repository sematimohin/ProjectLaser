Для обновления локальной версии репозитория на компьютере, необходимо выполнить следующие команды в терминале (командной строке) внутри папки с репозиторием:

1. git fetch: загрузит все изменения с репозитория, но не сложит их на рабочую копию проекта
2. git checkout [имя_ветки]: перейти на нужную ветку репозитория
3. git merge origin/[имя_ветки]: слить изменения с локальными файлами проекта

В результате выполнения этих команд вы получите актуальную версию репозитория на вашем компьютере.

Если вы хотите также загрузить новые файлы, которых нет на вашем компьютере, можно выполнить команду git pull. Она загрузит изменения с репозитория и автоматически сложит их на рабочую копию вашего проекта. Однако перед выполнением этой команды убедитесь, что все внесенные изменения зафиксированы и закоммичены на гитхабе.
